# ⚡ My Productivity Dashboard

A clean, modern **to-do list** with per-task **AI-powered motivational tips** generated by [Claude (Anthropic)](https://www.anthropic.com).

Built with **React + TypeScript + Vite** on the frontend and a **Vercel Serverless Function** as a secure backend proxy for the Anthropic API.

---

## Features

| Feature | Details |
|---|---|
| ✅ Add / complete / delete tasks | Instant UI, no page reload |
| 💾 Auto-save to localStorage | Tasks survive page refreshes |
| ✨ AI Tip button per task | Calls Claude to generate a fun motivational tip |
| 📊 Progress ring in header | Visual % completion indicator |
| 🗑 Clear completed | One-click bulk removal |
| 📱 Responsive | Works on mobile & desktop |
| 🔐 Secure API key | Key lives only in Vercel env, never in the browser |

## Quick Start (Local Development)

### Prerequisites

- [Node.js](https://nodejs.org) ≥ 18
- An [Anthropic API key](https://console.anthropic.com/settings/keys)
- [Vercel CLI](https://vercel.com/docs/cli) (for running serverless functions locally)

### 1 — Install dependencies

```bash
npm install
```

### 2 — Set up your API key

```bash
cp .env.example .env.local
# Edit .env.local and add your key:
#   ANTHROPIC_API_KEY=sk-ant-...
```

### 3 — Start the dev server

Use `vercel dev` so both the Vite frontend and the `/api` serverless function run together:

```bash
npx vercel dev
```

> Alternatively, if you only want to preview the UI without the AI feature: `npm run dev`

Open <http://localhost:3000> (Vercel dev) or <http://localhost:5173> (Vite only).

## Deployment (Vercel)

### Option A — GitHub + Vercel Dashboard (Recommended)

1. Push this repo to GitHub:

```bash
git init
git add .
git commit -m "feat: initial productivity dashboard"
git remote add origin https://github.com/YOUR_USERNAME/productivity-dashboard.git
git push -u origin main
```

2. Go to <https://vercel.com/new> → Import the `productivity-dashboard` repo.

3. In **Environment Variables**, add:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** `sk-ant-...` (your key from [console.anthropic.com](https://console.anthropic.com/settings/keys))

4. Click **Deploy**. Vercel will build, deploy, and give you a live URL.

### Option B — Vercel CLI

```bash
npx vercel --prod
# Follow the prompts; set ANTHROPIC_API_KEY when asked for env vars
```

## How the AI Tip Feature Works

```
Browser                  Vercel Serverless           Anthropic API
  │                            │                          │
  │── POST /api/generate-tip ──▶                          │
  │   body: { task: "..." }    │── messages.create() ────▶│
  │                            │◀─ { tip: "..." } ────────│
  │◀─ { tip: "..." } ──────────│                          │
  │                            │                          │
```

1. User clicks **✨ AI Tip** on a task.
2. The React frontend sends a `POST /api/generate-tip` request with the task text.
3. The serverless function (`api/generate-tip.js`) uses `@anthropic-ai/sdk` to call `claude-opus-4-6`.
4. The tip text is returned to the browser and displayed below the task.

The `ANTHROPIC_API_KEY` is **never** shipped to the browser — it only exists in the Vercel serverless environment.
